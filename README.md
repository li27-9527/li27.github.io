# li27 GitHub Pages

这是 li27 的个人资料与文件展示站，使用 GitHub Pages 和 Jekyll 构建。

## 在线维护

- 首页内容：编辑 `index.md`。
- 资料列表：由 `_materials/` 目录自动生成。
- 新增资料：在 `_materials/` 中创建 Markdown 文件。
- 附件文件：上传到 `assets/files/`，再从 Markdown 中链接。
- 图片资源：上传到 `assets/images/`。
- 维护入口：访问 `https://li27-9527.github.io/maintain/`。

提交到 `main` 分支后，GitHub Pages 会自动构建并发布。

## 快捷入口

- 新增资料页：https://github.com/li27-9527/li27-9527.github.io/new/main/_materials
- 上传附件：https://github.com/li27-9527/li27-9527.github.io/upload/main/assets/files
- 上传图片：https://github.com/li27-9527/li27-9527.github.io/upload/main/assets/images
- 编辑首页：https://github.com/li27-9527/li27-9527.github.io/edit/main/index.md

浏览器上传单文件限制为 25 MiB。更大的文件建议使用本地 git，超过 100 MiB 的文件应考虑 Git LFS。
